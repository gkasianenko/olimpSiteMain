Варианты подсветки объемных букв

Торцевая

Объемные буквы с подсветкой торцевой стороны. Возможна комбинация цветов.

Фронтальная

Объемные буквы из ПВХ с лицевой поверхностью из акрила. Могут быть белые, цветные, или с печатью.

Контражурная

Объемные буквы из ПВХ. Подсветка с задней стороны. Буквы крепятся на дистанционные держатели.

Пиксельная

Объемные буквы с открытыми светодиодами фронтальной стороны. Возможна комбинация цветов.
